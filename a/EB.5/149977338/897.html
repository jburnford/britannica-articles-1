ARIA, called Ariapolis, (Strabo): Now Herat, in Chorasan, set down in an ancient map as situated on the river Arias, which probably gave name to the country Aria. Arrian calls the river Arcios; Pliny, Arius; Ammian, Arios; now Heri, which runs by Alexandria, also called Alexandria Arion or Ariorum.
